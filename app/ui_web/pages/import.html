取込後に確認するポイント

  • 未知カラムや未知取引種別は取込ログに残るよ。
  • CSV は read-only import で、元ファイルは上書きしないよ。
  • 要確認件数が増えたら、JPY未評価や未知取引が混ざっていないか見てね。
  • 取込形式を直したあとに再取込する時は、先に「取引データをリセット」を押すと混ざりにくいよ。
